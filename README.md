# CSharpUsefulExtensions
This library is a collection of useful extension methods in C#.

## Methods

### StringBuilder methods:
* IsEmpty
* IsNullOrEmpty
* IsNullOrWhiteSpace

### Guid methods:
* IsEmpty

### DateTime methods:
* FirstDayOfTheMonth
* FirstDayOfTheMonthWithTime
* LastDayOfTheMonth
* LastDayOfTheMonthWithTime
